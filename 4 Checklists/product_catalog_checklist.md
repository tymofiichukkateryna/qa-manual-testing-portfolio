Checklist for product catalog	
Functional testing(positive)	Log in with valid credentials. Verify that User is redirected to the Products page and the product catalog is displayed. No error messages and blank page appear.
Functional testing(positive)	Log in with valid credentials. Open the Products page. Verify that all 6 products are displayed. Shouldn't be no missing products, duplicated products.
Functional testing(positive)	Log in with valid credentials. Open the Products page. Select sorting option (Name A–Z, Name Z-A, Price low–high, Price high-low) Verify that products reorder according to selected filter. No incorrect sorting order.
Functional testing(positive)	Log in with valid credentials. Click menu icon (☰) to open Side menu. Verify that Side menu appears.
UI/UX testing(positive)	Log in with valid credentials. Open the Products page. Verify that each product displays an image. Should NOT happen: broken image icon, missing image.
UI/UX testing(positive)	Log in with valid credentials. Open the Products page. Check product titles. Verify that product names are readable and correctly displayed. No overlapping text, typos, truncated names.
UI/UX testing(positive)	Log in with valid credentials. Open the Products page. Observe description under each product. Verify that descriptions are visible and readable. Should not be no empty descriptions, typos, layout overlap.
UI/UX testing(positive)	Log in with valid credentials. Open the Products page. Observe product grid layout. Verify that products are aligned properly. No overlapping elements or broken layout.
UI/UX testing(positive)	Log in with valid credentials. Open the Products page. Resize browser window. Verify that Layout adjusts correctly. Should not be hidden, cpopped and overlapping elements.
Security testing(negative)	Open inventory URL without authentication. Verify that User is redirected to login page. There is no unauthorized access.
Functional/UI/UX testing(positive)	Log in with valid credentials. Open the Products page. Check product prices. Verify that each product shows correct price with $ format. There are no missing price or incorrect format.
