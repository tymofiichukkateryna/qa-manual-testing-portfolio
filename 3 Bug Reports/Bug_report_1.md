| **Summary:**            | **User cannot log in when the username contains trailing spaces.**                                                                                                                                                                                                                                                                                        |
|-------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Priority: **          | High                                                                                                                                                                                                                                                                                                                                                      |
| **Environment**         |                                                                                                                                                                                                                                                                                                                                                           |
| **Device: **            | Desktop                                                                                                                                                                                                                                                                                                                                                   |
| **OS: **                | Windows 10 Pro (64-bit)                                                                                                                                                                                                                                                                                                                                   |
| **Browser:**            | Chrome 145.0.7632.160 (Official Build) (64-bit)                                                                                                                                                                                                                                                                                                           |
| **Network:**            | Wi-Fi                                                                                                                                                                                                                                                                                                                                                     |
| ****                    |                                                                                                                                                                                                                                                                                                                                                           |
| **Description**         |                                                                                                                                                                                                                                                                                                                                                           |
| ****                    | When a user enters a valid username with trailing spaces and a valid password, the system does not trim the spaces and treats the username as incorrect. As a result, the login attempt fails and an error message is displayed. The system should either trim trailing spaces automatically or handle them correctly to allow successful authentication. |
| **Preconditions:**      |                                                                                                                                                                                                                                                                                                                                                           |
| **·        **           | User is on login page https://www.saucedemo.com/                                                                                                                                                                                                                                                                                                          |
| **Steps to Reproduce:** |                                                                                                                                                                                                                                                                                                                                                           |
| **1.     **             | Enter username with trailing spaces.                                                                                                                                                                                                                                                                                                                      |
| **2.     **             | Enter valid password.                                                                                                                                                                                                                                                                                                                                     |
| **3.    **              | Click Login.                                                                                                                                                                                                                                                                                                                                              |
| **Expected Result:**    |                                                                                                                                                                                                                                                                                                                                                           |
| **·        **           | The system trims trailing spaces or processes the username correctly.                                                                                                                                                                                                                                                                                     |
| **·        **           | User is successfully logged in and redirected to the Products page.                                                                                                                                                                                                                                                                                       |
| **·        **           | User session becomes active.                                                                                                                                                                                                                                                                                                                              |
| **·        **           | No error message is displayed.                                                                                                                                                                                                                                                                                                                            |
| **Actual Result:**      |                                                                                                                                                                                                                                                                                                                                                           |
| **·        **           | Error message "Epic sadface: Username and password do not match any user in this service" is shown below the passworf field.                                                                                                                                                                                                                              |
| **·        **           | User remains on login page                                                                                                                                                                                                                                                                                                                                |
| **Repro rate:**         |                                                                                                                                                                                                                                                                                                                                                           |
| **·        **           | 100%                                                                                                                                                                                                                                                                                                                                                      |
| **Notes:**              |                                                                                                                                                                                                                                                                                                                                                           |
| **·        **           | The issue occurs when the username contains trailing spaces.                                                                                                                                                                                                                                                                                              |
| **·        **           | The system should automatically trim leading and trailing spaces during authentication to prevent login failures caused by minor input formatting issues.                                                                                                                                                                                                 |
| **·        **           | This behavior may negatively impact user experience because users often accidentally add spaces when copying credentials.                                                                                                                                                                                                                                 |
