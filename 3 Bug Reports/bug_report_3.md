| Description         |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
|                     | When a product is added to the cart, the button text changes from ""Add to cart"" to ""Remove"" as expected. However, after selecting ""Reset App State"" from the side menu, the cart counter icon disappears, indicating that the cart state has been reset. Despite this, the product button text still displays ""Remove"" instead of returning to ""Add to cart"".<br>This results in an inconsistent UI state because the application indicates that the cart is empty while the button suggests that the product is still in the cart. |
