| Priority:           | High                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| Environment         |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| Device:             | Desktop                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| OS:                 | Windows 10 Pro (64-bit)                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| Browser:            | Chrome 145.0.7632.160 (Official Build) (64-bit)                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| Network:            | Wi-Fi                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
|                     |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| Description         |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
|                     | The system allows users to complete the checkout process even when the shopping cart is empty. After opening the checkout page and filling in the required information, the user can proceed to the checkout overview and successfully finish the order. The system displays an order confirmation message despite no products being added to the cart.<br>This behavior is incorrect because checkout should only be possible when at least one product is present in the cart. |
| Preconditions:      |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| ·                   | User is on login page https://www.saucedemo.com/                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| Steps to Reproduce: |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| 1.                  | Enter a valid username and password and click login                                                                                                                                                                                                                                                                                                                                                                                                                              |
| 2.                  | Open the Cart page                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| 3.                  | Click Checkout                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| 4.                  | On the checkout form, enter First Name, Last Name, and Zip Code, then click Continue                                                                                                                                                                                                                                                                                                                                                                                             |
| 5.                  | Observe Checkout overview page with price total $0.00 and click finish                                                                                                                                                                                                                                                                                                                                                                                                           |
| 6.                  | Observe that the Checkout Complete page opens                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| Expected Result:    |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| ·                   | Checkout process should not proceed with an empty cart                                                                                                                                                                                                                                                                                                                                                                                                                           |
| ·                   | The user should remain on the cart page until at least one product is added.                                                                                                                                                                                                                                                                                                                                                                                                     |
| Actual Result:      |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| ·                   | Checkout form opens even when the cart is empty. After entering checkout information, the user can proceed to the Checkout Overview page with a total price of $0.00.                                                                                                                                                                                                                                                                                                            |
| ·                   | After clicking Finish, the Checkout Complete page opens with the message: "Thank you for your order! Your order has been dispatched, and will arrive just as fast as the pony can get there!"                                                                                                                                                                                                                                                                                    |
| Repro rate:         |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| ·                   | 100%                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| Notes:              |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| ·                   | The issue allows users to complete the checkout flow without adding any products to the cart.                                                                                                                                                                                                                                                                                                                                                                                    |
| ·                   | This may indicate missing validation for cart contents before initiating checkout.                                                                                                                                                                                                                                                                                                                                                                                               |
| ·                   | The system should validate that the cart contains at least one item before allowing the user to proceed with checkout.                                                                                                                                                                                                                                                                                                                                                           |
| ·                   | Potential impact: incorrect order records, incorrect analytics data, and poor user experience.                                                                                                                                                                                                                                                                                                                                                                                   |
