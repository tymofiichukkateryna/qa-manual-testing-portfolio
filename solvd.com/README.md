
## solvd.com Website Testing
# Project Description
This project contains manual testing documentation for the solvd.com website.
The testing was performed as part of a QA portfolio project to simulate a real testing process.
## Testing Scope
The following areas of the website were tested:
- Website navigation
- UI elements and layout
- Buttons and interactive elements
- Page transitions
- Content rendering
- Cookie policy page behavior
## Testing Types
- Exploratory testing
- Functional testing
- UI/UX testing
- Navigation testing
- Basic compatibility testing
## Test Environment
- Environment: Production
- Device: Desktop
- Operating System: Windows 10 Pro (64-bit)
- Browser: Google Chrome 
- Network: Wi-Fi
## Purpose of This Project
The goal of this project is to showcase practical QA skills such as:
- Writing test documentation
- Identifying usability and functional issues
- Reporting bugs clearly and professionally
