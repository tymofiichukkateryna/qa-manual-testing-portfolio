
# Test Plan
## Project Name
Website Solvd.com Testing
## Objective
- Verify that the website works according to expected functionality
- Identify defects in UI, navigation, and functionality
- Ensure that users can interact with the website elements without issues
- Validate correct behavior of links, buttons, and navigation elements"
## Scope
The following areas will be tested:
- Website navigation
- Links and buttons functionality
- UI/UX behavior
- Content loading
- Cookie policy page behavior
- Contact and support section buttons
## Features tested
- Contact section
- Home page
- Navigation menu
- Cookie Policy page
- Buttons and interactive elements
## Test Types
- Exploratory testing
- Functional testing
- UI/UX testing
- Navigation testing
- Basic compatibility testing
Testing is performed on the production version of the website.
## Test Environment
- Device: Desktop
- Browser: Google Chrome
- OS:  Windows 10 Pro (64-bit)
- Network: Wi-Fi
## Entry Criteria
Website is accessible. Test environment is stable.
## Exit Criteria
All critical defects are fixed.
